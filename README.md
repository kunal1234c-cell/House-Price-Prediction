# House-Price-Prediction
An end-to-end machine learning project for predicting house prices using Python and Scikit-learn. The project includes data preprocessing, exploratory data analysis, feature engineering, model training and evaluation using regression algorithm- Random Forest.

Project Overview 
The objective is to build a machine learning model capable of predicting house prices based on various housing features.

The project follows a complete machine learning pipeline:
Data Collection
Exploratory Data Analysis (EDA)
Data Preprocessing
Feature Engineering
Model Training
Model Evaluation
